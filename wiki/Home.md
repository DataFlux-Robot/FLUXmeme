# FLUXmeme Wiki

Welcome to the FLUXmeme wiki — the knowledge base for **DevReady assets**.

> **One robot. One `.flux` file. Body + mind + lifetime journal.**

FLUXmeme is a self-describing asset format for embodied AI. It upgrades
SimReady (physics-accurate 3D) into **DevReady** (physics + knowledge + agent +
lifetime journal), in one file, from cloud to MCU.
